# 塑料光谱识别系统

本项目是一个基于 PyQt5 的桌面应用程序，用于通过串口连接光谱设备，接收光谱数据，并利用分类模型识别塑料类型。同时，它还提供了光谱数据的实时显示和保存功能。

## 环境要求

- **Python 版本**: 推荐使用 Python 3.12
- **Python 管理器**: 使用 `uv`

### 安装依赖

在 `识别算法` 目录下执行：

```bash
uv sync
```

## 运行与命令

所有命令均在 `识别算法` 目录下执行。

### 1. 启动图形界面

```bash
uv run main.py
```

### 2. 训练并导出模型

```bash
uv run knn.py
```

当前训练脚本会自动比较多种候选模型，而不是固定使用 KNN。

- `knn_baseline`: 原始方案，`StandardScaler + Normalizer + KNN`
- `ratio_random_forest`: 基于通道比值特征的 `RandomForest`
- `ratio_extra_trees`: 基于通道比值特征的 `ExtraTrees`

训练完成后会自动选择交叉验证分数最高的模型，并导出：

- `classifier.pkl`: 当前默认加载的模型文件
- `knn.pkl`: 兼容旧版界面的副本

### 3. 合并 CSV 数据

```bash
uv run tool.py
```

## `ui.py` 使用说明

程序启动后，会显示主界面。

### 2. 串口设置

- **串口选择**: 从"串口:"下拉列表中选择您的光谱设备所连接的串口号。如果列表中没有出现您的串口，请尝试点击"刷新串口"按钮。
- **波特率**: 根据您的设备手册选择正确的波特率（默认为 115200）。
- **数据位、校验位、停止位**: 根据您的设备要求进行设置（通常默认为 8-N-1，即8个数据位，无校验，1个停止位）。
- **连接/断开设备**: 设置完成后，点击"连接设备"按钮连接设备。连接成功后，按钮文本变为"断开连接"，此时可以进行数据收发。再次点击可断开连接。

### 3. 数据收发与显示

- **接收显示**: 连接串口后，从设备接收到的原始数据会记录在"实时日志"中。温度数据 `tempF[value]` 会同步更新到顶部温度指标。
- **发送区**: 您可以在下方的文本框中输入指令，然后点击"发送指令"按钮将其通过串口发送出去。输入框支持把 `sc/r/n` 或 `sc\\r\\n` 自动转换为真实的回车换行。

### 4. 塑料识别

- **开始识别**: 串口连接后，点击"开始识别"按钮，程序会向下位机发送扫描指令 `sc\r\n`，并等待本次扫描返回的六通道数据。
- **数据累积与预测**: 程序会从串口数据中提取 `R,S,T,U,V,W` 六个通道；当六通道收齐后，会立即加载已训练的模型并执行一次预测。
- **识别结果**: 预测得到的塑料类型会显示在"识别结果:"旁边的文本框中。

### 5. 训练模式数据采集

- **开始采集**:
  - 切换到"训练模式"后，点击"开始采集"。
  - 每次开始采集或完成一轮完整扫描后，程序都会自动向下位机发送 `sc\r\n` 触发下一次扫描。
  - 程序会在数据目录下自动创建或复用 `train_data.csv`，后续标注成功的样本会持续追加到该文件。
- **完整性判定**:
  - 一次有效扫描必须完整包含 `R,S,T,U,V,W` 六个通道。
  - 如果在六通道收齐之前出现重复通道，本次扫描会被判定为不完整并作废，然后重新开始累计当前次扫描。
- **重复采集与求均值**:
  - "采集次数"表示同一组样本需要累计多少次完整扫描。
  - 当程序收满目标次数后，会对这几次扫描的六通道数值逐列求平均，得到一组最终训练特征。
- **手动标注与入库**:
  - 求平均后会弹出标签选择框。
  - 只有点击"确认标注"后，这组均值数据才会追加到 `train_data.csv`。
  - 如果取消标注，则当前组数据直接作废，不写入训练集。

### 6. 数据文件说明

- 默认训练数据文件为 `data/train_data.csv`。
- 文件格式保持与 `knn.py` 一致：前 6 列为光谱特征，最后 1 列为标签，不写表头。
- 如果您在系统设置中修改了数据目录，训练模式会把新采集的样本追加到新目录下的 `train_data.csv`。

### 7. 注意事项

- 确保 `knn.py` 文件以及训练生成的模型文件 `classifier.pkl` 或 `knn.pkl` 与 `ui.py` 在同一目录下。
- 如果使用了 `CandyWindow` (如 `ui.py` 末尾所示)，请确保 `icon.png` 文件存在，或修改/移除图标设置。
- 光谱数据的键（R,S,T,U,V,W）是硬编码在 `EXPECTED_SPECTRUM_KEYS` 中的，如果您的设备使用不同的键，需要相应修改。
